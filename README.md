## Como Correr el proyecto

1. Clona el Repositorio:
```bash
git clone
https://github.com/juanjot08/ElectivaII_ClonTinder_LastDance.git
```
2. Instalar dependencias
```bash
npm install
```

3. Correr proyecto
```bash
npm run dev
```
